#main

i를 0으로 시작해서 5보다 작은 동안 반복하고, 매 회차마다 1씩 증가시킨다. 결과적으로 "안녕"이 딱 5번 출력된다. i < 5 조건을 i < 3으로 바꾸면 3번만 출력되는 식으로, 숫자만 바꾸면 반복 횟수를 자유롭게 조절할 수 있다

#main2

이름은 똑같이 add인데, 전달하는 값의 개수나 타입이 다르면 자바가 알아서 맞는 메소드를 골라 실행한다. 덕분에 addTwoInts, addThreeInts 같이 이름을 여러 개 만들 필요 없이 add 하나로 통일할 수 있어서 코드가 깔끔해진다.

#main3

Student가 Person을 상속받아 introduce()를 자기 방식으로 재정의(오버라이딩)한 것이다. 핵심은 세 번째 줄인데, Student 객체를 Person 타입 변수에 담아도 실제로 실행될 때는 Student의 메소드가 호출된다. 이게 다형성이고, 이를 통해 타입은 통일하되 동작은 각자 다르게 만들 수 있다.
